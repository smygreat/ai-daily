## 一、论文速递

### LLM 架构 / 推理

- **Layer Equivalence Is Not a Property of Layers Alone** — arXiv
  - **核心 Idea**: 模型压缩时判断"两个 Transformer 层是否等价"有两种测试方法——Replace（替换某层）和 Interchange（交换两层位置）——它们经常给出不同答案。选错测试方法可能导致误剪掉关键层。
  - **方法亮点**: 在 Pythia (410M/1.4B) 训练全程和 Qwen3-8B / Llama-3.1-8B 上系统对比。发现同一 checkpoint 下，Interchange 引导的层删除比 Replace 安全数倍；但 metric gap 的大小和删除安全性并不一一对应。
  - **关键发现**: Qwen3-8B 出现"发散模式"——两种协议给出的安全剪枝范围差异极大。Llama-3.1-8B 两种协议虽 KL 差异大，但剪枝代价却接近。
  - **实践建议**: 在剪枝或合并前，两个 swap-KL 都要跑一遍，只需要无标签前向传播即可。
  - **对比**: 相比传统的层重要性评分（如 Angular/WANDA），这篇直接指出"你问问题的方式决定了你会删除什么层"——是对剪枝方法论本身的元反思。
  - **值得关注**: ⭐⭐⭐⭐

### Agent

- **FORGE: Self-Evolving Agent Memory With No Weight Updates via Population Broadcast** — arXiv
  - **核心 Idea**: 不更新模型权重，让 Agent 通过"群体广播"机制自我进化记忆。多个 Agent 实例并行运行，反思失败的轨迹，生成文本化的启发规则和示例，最佳记忆在群体间传播。
  - **方法亮点**: 在 CybORG CAGE-2 网络防御任务（30 步 POMDP）上，4 个 LLM 系列（Gemini-2.5-Flash-Lite, Grok-4-Fast, Llama-4-Maverick, Qwen3-235B）的零样本奖励都是严重负值。FORGE 将平均回报提升 1.7-7.7 倍，严重失败率降至 ~1%。
  - **关键发现**: ① 群体广播是核心机制，graduation 主要节省算力而非提升性能；② Examples（示例）对 3/4 模型效果最好，Rules（规则）有最优成本-可靠性比（少 ~40% tokens）；③ 弱模型反而受益更大——FORGE 可能缩小能力差距而非放大。
  - **对比**: vs Reflexion（单实例自反思）→ 提升 29-72%；vs 传统 fine-tuning → 不需要任何梯度更新，纯 prompt 注入。
  - **值得关注**: ⭐⭐⭐⭐⭐

- **Autonomous LLM-Guided Tree Search for Multi-Pathogen Disease Forecasting** — arXiv
  - **核心 Idea**: 用 LLM 自主驱动树搜索，自动生成、评估、优化传染病预测模型。在 2025-2026 美国呼吸道疾病季实时测试中，机器生成的模型集成持续匹配或超越了 CDC 人类专家手动调优的黄金标准。
  - **方法亮点**: 成功处理 RSV 的"冷启动"数据稀缺场景。自动 judge-in-the-loop 保证科学结构保真度；log-scale 距离度量的选择防止了 reward hacking。
  - **启发**: "AI 做科研"不是口号——这是 AI 自主完成方法论发现→代码生成→评估优化的完整闭环，在真实公共卫生场景中跑赢了人类专家团队。
  - **值得关注**: ⭐⭐⭐⭐

### CV / 多模态

- **IVGT: Implicit Visual Geometry Transformer** — arXiv
  - **核心 Idea**: 把 3D 几何表示为隐式连续函数而非离散点云，从无姿态多视角图像直接重建连续、连贯的 3D 表面。
  - **方法亮点**: 在规范坐标系中学习连续神经场景表示，支持任意 3D 位置的空间查询（SDF + 颜色）。多数据集联合优化 + 2D 监督 + 3D 几何正则化。
  - **对比**: vs DUSt3R/MASt3R —— IVGT 不做逐像素点图回归，而是建模连续几何场，几何连续性更好。
  - **值得关注**: ⭐⭐⭐

### RL / 机器人

- **VLA-AD: Offline Semantic Guidance for VLA Policy Distillation** — arXiv
  - **核心 Idea**: 用 VLM 作为离线"语义教师"，将 7B 参数的视觉-语言-动作（VLA）大模型压缩到 158M 参数的轻量学生模型——44 倍缩小，性能差距仅 0.27%。
  - **方法亮点**: 不只是模仿动作，而是加入任务阶段锚点和多帧操作方向描述作为辅助语义信号。学生模型推理速度 12.5Hz（RTX 4090），比 OpenVLA-7B 快 3.28 倍。对教师模型输出的噪声动作（如高频夹爪抖动）更鲁棒。
  - **对比**: vs 传统行为克隆 → VLA-AD 增加的语义监督让学生学到"为什么做"而不只是"怎么做"。
  - **值得关注**: ⭐⭐⭐⭐

### AI 社会影响

- **AI-Mediated Communication Can Steer Collective Opinion** — arXiv
  - **核心 Idea**: AI 不仅影响人机交互中的个体观点，当它介入*人与人的沟通*（如 LinkedIn 润色帖子、X "解释此帖"）时，其偏见可以通过社交网络被放大，最终改变群体舆论方向。
  - **方法亮点**: 实证 + 理论双层验证。实证显示多个主流 LLM 在编辑争议话题文本时引入方向性偏见（如偏向枪支管制、反对无神论）；建立数学模型证明偏见通过网络传播可达均衡偏移。对 X 的 Grok "解释此帖"功能进行审计，发现 Pro-Life 偏向可追溯到具体设计选择。
  - **影响**: 这篇直接关联到欧盟正在推进的 AI 监管立法。它回答了"AI 审查/操纵舆论"到底是阴谋论还是真实威胁——答案是后者，且有数学证明。
  - **值得关注**: ⭐⭐⭐⭐⭐

### 基础设施

- **Designing Datacenter Power Delivery Hierarchies for the AI Era** — arXiv (Microsoft Azure)
  - **核心 Idea**: 2027 年 AI 加速器部署将接近 1MW/机架，现有的数据中心供电架构会"搁浅"大量电力资源。核心问题不是装了多少兆瓦，而是能部署多少有效算力。
  - **方法**: 结合 GPU/计算/存储部署投影模型 + Azure 生产数据，评估供电拓扑、部署粒度、放置策略、电力超配的综合影响。
  - **值得关注**: ⭐⭐⭐（从业者参考，非学术突破）
